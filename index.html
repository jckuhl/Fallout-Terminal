Fallout Terminal

Fallout Terminal emulates a terminal from the Fallout series, specifically Fallout 3 and 4. Try to guess the password before you run out of attempts!

How to play:

The object of the game is to guess the password. One of the ten or so words on the screen will be the password. You have five chances to guess. If you pick the wrong one, you lose a chance, however it will offer you clues by telling you which letters in the word match. Each matching letter must be the same letter in the same position in the word

Example:

If the password is "fate" and I chose "mute", then it will reveal I have two matches because the "t" and the "e" are in the same place in both words. However, even if the same letters are present, such as in "evil" and "vile", there are no matches because none of the letters are in the same spot.

You can also scan for any brackets in the code. If there are any matching brackets on the same line that don't contain a word, they'll help you. They can remove a dud or, if you're really lucky, reset your attempts.

There are 4 difficulties:

  • Novice: words are 4-5 letters long
  • Advanced: words are 6-8 letters long
  • Expert: words are 9-10 letters long
  • Master: words are 11-12 letters long
